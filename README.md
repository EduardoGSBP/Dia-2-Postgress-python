🚀 Projeto: Pipeline de Dados com Python + PostgreSQL

Este projeto tem como objetivo desenvolver um pipeline de dados completo, integrando Python, SQL e PostgreSQL, com foco em práticas reais de Engenharia de Dados, como carga incremental, organização de código e manipulação eficiente de dados.

📌 Objetivo

Construir uma estrutura capaz de:

Conectar com banco de dados PostgreSQL de forma eficiente
Realizar ingestão e transformação de dados
Implementar cargas incrementais
Trabalhar com diferentes formatos de arquivos
Criar um fluxo organizado e reutilizável
🛠️ Tecnologias utilizadas
Python
SQLAlchemy
PostgreSQL
Pandas
⚙️ Funcionalidades do Projeto
🔗 Conexão com Banco de Dados
Conexão com PostgreSQL utilizando SQLAlchemy
Abstração da conexão para facilitar reutilização
🔄 Carga Incremental de Dados

Implementações realizadas:

Carga incremental baseada em ano
Refatoração do código para tornar o processo reutilizável
Carga incremental considerando os últimos 3 meses
📥 Integração SQL + Python
Importação de consultas SQL diretamente para o ambiente Python
Execução de queries e transformação dos dados em DataFrames
💾 Manipulação de Arquivos

Leitura de arquivos:

JSON
CSV
Parquet
Excel

Exportação de dados:

JSON
CSV
Parquet
Excel
